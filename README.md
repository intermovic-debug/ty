# Meritz SOXL Automation

Sanitized transfer copy for a local Meritz HTS SOXL automation experiment.

Important safety defaults:

- Run in `dry_run` first.
- Do not commit live account data, passwords, HTS logs, screenshots, state files, or calibrated coordinates.
- Recalibrate all coordinates on the home PC.
- Keep real trading disabled until dry-run behavior is verified with tiny quantities.

Start with `HOME_PC_SETUP.md`.
